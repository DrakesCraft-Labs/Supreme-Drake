## ⚡ Integración Nativa Rust (Opción A: Modelo Híbrido Cero-Riesgo)

`Supreme-Drake` incluye el puente Panama FFM **`RustNativeBridge`** para delegar la aceleración de los multiplicadores de tarjetas de velocidad/energía y tickers de generadores cuánticos directamente a la librería `slimefun_ffi` en Rust.

---

## 🛠️ Compilación

```bash
mvn clean package
```
